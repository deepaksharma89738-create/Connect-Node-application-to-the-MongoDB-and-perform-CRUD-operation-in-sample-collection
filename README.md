# Connect-Node-application-to-the-MongoDB-and-perform-CRUD-operation-in-sample-collection
Connect a Node.js app to MongoDB using Mongoose. Define a schema and model, then perform CRUD operations: create documents with `save()`, read using `find()`, update with `updateOne()` or `findByIdAndUpdate()`, and delete using `deleteOne()`. This enables efficient data handling in a sample collection.
